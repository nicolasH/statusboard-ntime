Status Board widgets

Simple widget(s) for Panic's Status Board iPad application. Embeddable using the "Do-It-Yourself (HTML)" and "Table" widgets

  • nTime, a compact time widget with simple time offset / timezone support. (blog post with screenshot)
  • appleDev a table-based widget showing the Apple Developer System status page in a statusboard table. You can select which sub-systems you want to see on your widget.
